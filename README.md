# Palash Mulchandani — Portfolio

🔗 **Live:** [palashmulchandani.github.io](https://palashmulchandani.github.io)

Personal portfolio site — a single-page, dark-themed showcase of my AI engineering projects, built entirely from scratch with vanilla HTML, CSS, and JavaScript.

## What's inside

- **Hero** — animated typing role rotation, live SVG speedometer stats (AI products shipped, live deployments)
- **About** — background, current focus, education snapshot
- **Experience** — timeline of leadership and independent development work
- **Education** — SVVV Indore, Woxsen University, school records
- **Projects** — Nexus, PrepSense, CalmCampus, BeyondBooks, each with live/GitHub links and full tech stack
- **Metrics** — canvas-drawn animated gauges (JEE Mains percentile, Class 10 & 12 scores) that loop continuously
- **Skills** — categorized tech stack (AI/ML, Backend, Frontend, DevOps)
- **Currently** — what I'm actively building, studying, and targeting right now
- **Achievements** — hackathons, certifications, academic recognition
- **Contact** — direct links to email, LinkedIn, GitHub

## Tech stack

- Pure HTML5, CSS3, vanilla JavaScript — no frameworks, no build step
- Canvas API for animated speedometer gauges
- Intersection Observer for scroll-triggered reveals
- [GoatCounter](https://www.goatcounter.com) for privacy-friendly analytics

## Deployment

Hosted on GitHub Pages, served directly from this repo's `main` branch at the root.
